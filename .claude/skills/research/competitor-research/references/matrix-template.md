## Usage notes

### When to use matrix vs. deep dive
- **Matrix**: Quick overview, positioning exercise, initial discovery
- **Deep dive**: Battlecard prep, detailed competitive strategy, specific threat analysis

### Dimension priority for matrix
Focus on these 5 core dimensions:
1. Company (funding, team, valuation)
2. Product (core capability, differentiator)
3. ICP (who they serve)
4. Pricing (model and price points)
5. Reviews (ratings and sentiment)

### Handling incomplete data
- Use "—" for truly unavailable data
- Use "Unknown" when data may exist but wasn't found
- Add to Data Gaps table with follow-up action

### Keeping matrices readable
- Max 6 competitors per matrix
- Use abbreviations if column headers get long
- Add footnotes for nuanced data points
