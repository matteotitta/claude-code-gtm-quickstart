# Channel Playbooks

Detailed playbooks for each content channel: basic channels (Blog/Newsletter, LinkedIn) and bonus channels (Podcast/YouTube, Events, X.com, Instagram/TikTok).

---

## Channel Hierarchy

### Tier 1 — Basic (Always Recommend)

| Channel | Why Essential | Primary Use |
|---------|---------------|-------------|
| **Blog/Newsletter** | Owned authority, SEO, email list | Long-form educational content |
| **LinkedIn** | Primary B2B distribution | Amplification, engagement |

### Tier 2 — Anchor (If Capacity Allows)

| Channel | When to Include | Primary Use |
|---------|-----------------|-------------|
| **Podcast** | Founder available, relationship-driven sales | Content engine, relationships |
| **YouTube** | Visual product, tutorial-heavy, long-form | Video authority, search |

### Tier 3 — Amplification (Based on Context)

| Channel | When to Include | Primary Use |
|---------|-----------------|-------------|
| **X.com** | Tech/startup audience, real-time commentary | Reach, engagement |
| **Events** | High-touch sales, budget available | Conversion, relationships |
| **Instagram/TikTok** | Employer branding, younger demographic | Culture, talent |

## Basic Channels

### Blog/Newsletter

#### Overview

| Attribute | Specification |
|-----------|---------------|
| **Effort level** | Medium-High |
| **Primary pillar** | Educational (60%), Promotional (30%), Personal (10%) |
| **Primary metric** | Subscribers, open rate |
| **Secondary metrics** | Traffic, time on page, click-through |
| **Recommended frequency** | Weekly or bi-weekly |
| **Tooling options** | Substack, Beehiiv, ConvertKit, Ghost |

#### Content Types

| Type | Description | Frequency |
|------|-------------|-----------|
| **Deep-dive articles** | Long-form educational pieces | Weekly |
| **Podcast transcripts** | Expanded transcripts from episodes | If anchor is podcast |
| **Framework introductions** | New frameworks/methodologies | Monthly |
| **Curated roundups** | Industry news and commentary | Weekly optional |
| **Case study features** | Detailed customer stories | Monthly |

#### Newsletter Structure Template

```
Subject: [Benefit-driven subject line] | [Newsletter Name]

---

[Personal opening - 2-3 sentences]

---

## This Week's Focus: [Topic]

[3-5 paragraphs of value]

### Key Takeaways:
→ [Takeaway 1]
→ [Takeaway 2]
→ [Takeaway 3]

---

## Quick Links

• [Resource 1]
• [Resource 2]
• [CTA - if promotional]

---

[Sign-off]
[Name]
```

#### Naming Conventions

| Pattern | Example | Best For |
|---------|---------|----------|
| **[Topic] + [Format]** | "Data Delivery Digest" | Educational focus |
| **[Company] + [Format]** | "The Linear Changelog" | Company-centric |
| **[Verb] + [Topic]** | "Build with AI" | Action-oriented |
| **[Audience] + [Benefit]** | "Founder Insights" | Persona-specific |

#### Success Metrics

| Metric | Good | Great | Excellent |
|--------|------|-------|-----------|
| **Open rate** | 30%+ | 40%+ | 50%+ |
| **Click rate** | 3%+ | 5%+ | 8%+ |
| **Subscriber growth** | 5%/month | 10%/month | 20%/month |
| **Unsubscribe rate** | <1% | <0.5% | <0.3% |

### LinkedIn

#### Overview

| Attribute | Specification |
|-----------|---------------|
| **Effort level** | Low-Medium per post |
| **Primary pillar** | Mixed (all pillars) |
| **Primary metric** | Engagement rate, follower growth |
| **Secondary metrics** | Profile views, DMs, post impressions |
| **Recommended frequency** | 5x/week minimum |
| **Tooling options** | Native, Opus (for clips), Canva (graphics) |

#### Content Formats

| Format | Description | Engagement Level | Effort |
|--------|-------------|------------------|--------|
| **Text posts** | Pure text, vertical format | Highest | Low |
| **Carousels** | Multi-slide educational content | High | Medium |
| **Video clips** | Short-form video (30-90 sec) | High | Medium |
| **Polls** | Question with options | Medium | Low |
| **Image posts** | Single image with caption | Medium | Low |
| **Document posts** | PDF uploads | Medium | Medium |
| **Long-form articles** | LinkedIn articles | Low | High |

#### Best Practices

**Posting:**
- Post 5x/week minimum for consistent growth
- Best times: 8-10am and 12-2pm (audience timezone)
- Engage 30 min before and after posting
- Reply to every comment within 2 hours

**Format:**
- Vertical text with line breaks (no walls of text)
- Hook in first line (stops the scroll)
- Arrows (→) for lists
- Short sentences (12-15 words average)
- No emojis in hooks
- CTA at end (question or action)

**Don't:**
- Use link in main post (kills reach) — put in comments
- Post generic content (fails 100 Posts Test)
- Ignore comments
- Post walls of text
- Over-use emojis

#### Weekly Cadence Template

| Day | Content Type | Pillar | Example |
|-----|--------------|--------|---------|
| Monday | Hot take / Opinion | Personal | "Unpopular opinion: [X] is overrated" |
| Tuesday | Educational guide | Educational | "5 [patterns] every [role] needs" |
| Wednesday | Video clip (from podcast) | Educational/Promotional | Customer interview highlight |
| Thursday | Case study / Result | Promotional | "[Customer] achieved [result]" |
| Friday | Carousel / Framework | Educational | "The [framework] framework" |

#### Hook Formulas

| Type | Formula | Example |
|------|---------|---------|
| **Data/Number** | [Number] + [unexpected outcome] | "I've created 1600+ competitor pages with AI." |
| **Contrarian** | [Common belief] is wrong. | "Your messaging is not 'fine'. It's static." |
| **Story** | My [person/experience] showed me [insight]. | "My wife is a localization manager..." |
| **Challenge** | Most [audience] do [wrong thing]. | "Most teams blame tools for bad GTM." |
| **Question** | Still [outdated behavior]? | "Still 'playing' with ChatGPT?" |
| **Outcome** | [Person] did [impressive thing]. | "A client cut reporting from 5 hours to 20 minutes." |

#### Success Metrics

| Metric | Good | Great | Excellent |
|--------|------|-------|-----------|
| **Engagement rate** | 2%+ | 4%+ | 6%+ |
| **Follower growth** | 3%/month | 5%/month | 10%/month |
| **Profile views** | +10%/week | +25%/week | +50%/week |
| **Inbound DMs** | 5/week | 15/week | 30/week |

## Bonus Channels

### Podcast / YouTube (Anchor Content)

#### Overview

| Attribute | Specification |
|-----------|---------------|
| **Effort level** | High (but creates derivative content) |
| **Primary pillar** | Educational (50%), Personal (30%), Promotional (20%) |
| **Primary metric** | Subscribers, downloads/views |
| **Secondary metrics** | Retention, engagement, referral source |
| **Recommended frequency** | Weekly (minimum bi-weekly) |
| **Tooling options** | Zoom, Riverside, Descript, Opus |

#### Format Options

| Format | Description | Best For | Effort |
|--------|-------------|----------|--------|
| **Interview** | Guest conversations | Relationships, authority | Medium |
| **Solo** | Single host educational | Thought leadership | Medium |
| **Co-host** | Two-person discussion | Variety, chemistry | Medium |
| **Panel** | Multiple guests | Event-style, debates | High |
| **Q&A** | Audience questions | Engagement, community | Low |

#### Content Cascade

**From 1 podcast episode:**
```
1. Full episode (YouTube/Podcast platforms)
         ↓
2. Blog post (transcript + expansion)
         ↓
3. LinkedIn posts x5 (key quotes, insights)
         ↓
4. Short clips x3 (30-90 sec highlights)
         ↓
5. Quote graphics x2 (shareable images)
         ↓
6. Newsletter edition
```

**Multiplier: 1 episode → 12+ derivative pieces**

#### Episode Structure (Interview)

```
[0:00 - 2:00]   HOOK: Teaser of best moment
[2:00 - 5:00]   INTRO: Guest introduction, context
[5:00 - 15:00]  BACKSTORY: Guest journey, experience
[15:00 - 30:00] MEAT: Main topic deep-dive
[30:00 - 40:00] TACTICAL: Actionable takeaways
[40:00 - 45:00] CLOSING: Key insights, CTA
```

#### Guest Selection Criteria

| Type | Value | Example |
|------|-------|---------|
| **Customers** | Social proof, use cases | "7-Eleven data team" |
| **Industry experts** | Authority transfer | "Former Gartner analyst" |
| **Practitioners** | Tactical credibility | "Sr. Data Engineer at [company]" |
| **Founders** | Peer network | "CEO of [complementary tool]" |
| **Analysts** | Market perspective | "Partner at [VC firm]" |

#### Success Metrics

| Metric | Good | Great | Excellent |
|--------|------|-------|-----------|
| **Downloads/ep** | 500+ | 1,000+ | 5,000+ |
| **Subscriber growth** | 5%/month | 10%/month | 20%/month |
| **Avg retention** | 40%+ | 60%+ | 75%+ |
| **Guest conversion** | 10% become customers | 20% | 30% |

### Events

#### Overview

| Attribute | Specification |
|-----------|---------------|
| **Effort level** | High |
| **Primary pillar** | Promotional (50%), Educational (40%), Personal (10%) |
| **Primary metric** | Registrations, attendance |
| **Secondary metrics** | Follow-up meetings, pipeline generated |
| **Recommended frequency** | Monthly webinar, quarterly major |
| **Tooling options** | Zoom, Demio, Livestorm, Hopin |

#### Event Types

| Type | Format | Audience | Effort | Conversion |
|------|--------|----------|--------|------------|
| **Webinars** | Live presentation + Q&A | Prospects | Medium | Medium |
| **Workshops** | Hands-on learning | Engaged prospects | High | High |
| **Conferences** | Speaking/sponsoring | Broad awareness | High | Low |
| **Roundtables** | Small group discussions | High-value prospects | Medium | Very High |
| **Office hours** | Open Q&A | Existing users | Low | Retention |
| **Customer events** | User community | Customers | High | Expansion |

#### Webinar Best Practices

**Pre-event:**
- Promote 2-3 weeks in advance
- Email sequence: Announce → Remind → Day-before → 1-hour before
- LinkedIn promo: 3-5 posts leading up
- Clear value proposition in title

**During:**
- Start on time (don't wait for stragglers)
- Engaging visuals, not walls of text
- Q&A throughout, not just at end
- Clear CTA at end

**Post-event:**
- Send recording within 24 hours
- Follow up no-shows with recording
- Segment attendees by engagement
- Fast follow-up for high-intent signals

#### Success Metrics

| Metric | Good | Great | Excellent |
|--------|------|-------|-----------|
| **Registration rate** | 3% of invites | 5% | 8% |
| **Attendance rate** | 35% of registered | 45% | 55% |
| **Engagement score** | 50%+ active | 65% | 75% |
| **Meeting conversion** | 10% of attendees | 20% | 30% |

### X.com (Twitter)

#### Overview

| Attribute | Specification |
|-----------|---------------|
| **Effort level** | Low-Medium |
| **Primary pillar** | Personal (40%), Educational (40%), Promotional (20%) |
| **Primary metric** | Followers, engagement |
| **Secondary metrics** | Impressions, profile clicks |
| **Recommended frequency** | 3-5x/day minimum |
| **Tooling options** | Native, Buffer, Hypefury |

#### When to Include X.com

**Good fit:**
- Tech/developer audience
- Real-time commentary matters
- Founder is comfortable with platform
- Product has tech community buzz

**Not a fit:**
- Enterprise-only audience
- No one on team active on X
- Regulated industry
- Limited bandwidth

#### Content Types

| Type | Description | Best For | Frequency |
|------|-------------|----------|-----------|
| **Hot takes** | Short opinions | Visibility | Daily |
| **Threads** | Multi-tweet educational | Deep engagement | 2-3x/week |
| **Replies** | Engage in conversations | Community | Constant |
| **Quote tweets** | Commentary on others | Visibility | Daily |
| **Links** | Share content | Traffic | Sparingly |

#### X.com vs LinkedIn Differences

| Dimension | X.com | LinkedIn |
|-----------|-------|----------|
| **Tone** | Casual, raw | Professional, polished |
| **Length** | Shorter, punchier | Longer, structured |
| **Frequency** | 3-5x/day | 1x/day |
| **Engagement** | Real-time | Delayed |
| **Links** | OK in tweet | Kill in main post |

### Instagram / TikTok

#### Overview

| Attribute | Specification |
|-----------|---------------|
| **Effort level** | Medium-High |
| **Primary pillar** | Organizational (50%), Personal (30%), Educational (20%) |
| **Primary metric** | Followers, views |
| **Secondary metrics** | Engagement, saves, shares |
| **Recommended frequency** | 3-5x/week |
| **Tooling options** | Native, Canva, CapCut, Opus |

#### When to Include Instagram/TikTok

**Good fit:**
- Employer branding priority
- Hiring younger talent
- Culture is a differentiator
- Visual product/process

**Not a fit:**
- B2B enterprise only
- No employer brand need
- Limited video capability
- Regulated industry

#### Content Types

| Type | Platform | Description | Best For |
|------|----------|-------------|----------|
| **Reels** | Instagram | Short-form video | Culture, tutorials |
| **Stories** | Instagram | Ephemeral content | Behind-scenes |
| **TikToks** | TikTok | Trending format video | Culture, reach |
| **Carousels** | Instagram | Multi-image educational | Mini-tutorials |
| **Static posts** | Instagram | Single image + caption | Announcements |

#### Employer Brand Content Ideas

- "Day in the life" series
- Office/remote workspace tours
- Team introductions
- Culture moments (celebrations, rituals)
- "How we [process]" behind-scenes
- Employee takeovers
- Hiring announcement creatives

## Channel Selection Decision Tree

```
START
  │
  ├─ Do you need owned content for SEO/email?
  │    └─ YES → Blog/Newsletter ✓ (Basic)
  │
  ├─ Is B2B your primary audience?
  │    └─ YES → LinkedIn ✓ (Basic)
  │
  ├─ Is founder available 2-4 hrs/week for content?
  │    ├─ YES → Podcast/YouTube ✓ (Anchor)
  │    └─ NO → Skip, focus on written
  │
  ├─ Do you have budget for events?
  │    └─ YES → Events ✓ (Bonus)
  │
  ├─ Is audience tech/startup heavy?
  │    └─ YES → X.com ✓ (Bonus)
  │
  ├─ Is employer branding a priority?
  │    └─ YES → Instagram/TikTok ✓ (Bonus)
  │
END
```
