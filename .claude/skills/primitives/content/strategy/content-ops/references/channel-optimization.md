## LinkedIn optimization (360 Brew algorithm, Jan 2026)

### Algorithm signals (ranked by weight)

| Signal | Weight | What It Means |
|--------|--------|--------------|
| **Saves** | 5x likes | Content worth bookmarking gets massive distribution boost |
| **Comments (quality)** | 3-4x likes | Thoughtful comments from your network > "Great post!" from strangers |
| **Dwell time** | High | Longer time spent reading = algorithm keeps showing it |
| **Shares** | 2x likes | Re-shares to feed or DMs signal high value |
| **Likes** | 1x (baseline) | Lowest-weight engagement signal |
| **Profile visits** | Indirect | Content that drives profile visits signals relevance |

### Optimization checklist

- [ ] **Write for saves:** Include frameworks, templates, checklists, or reference material worth bookmarking
- [ ] **Optimize for dwell time:** Longer posts with substance outperform short ones. Use whitespace, formatting, and narrative to keep people reading
- [ ] **No external links in body:** Links reduce reach 25-40%. Put links in first comment
- [ ] **Consistent posting times:** Pick 2-3 time slots and stick to them. Algorithm rewards consistency
- [ ] **Prime first 90 minutes:** The first 90 minutes after posting determine distribution. Be available to respond to comments
- [ ] **Don't edit after publishing:** Editing resets the algorithm. Get it right before posting
- [ ] **Native formats preferred:** Documents (carousels), native video, and text outperform link posts

### Best posting times (B2B)

| Day | Time Slots | Notes |
|-----|-----------|-------|
| Tuesday | 8-9 AM, 2-3 PM | Highest B2B engagement days |
| Wednesday | 8-9 AM, 12-1 PM | Strong midweek performance |
| Thursday | 8-9 AM, 2-3 PM | Highest B2B engagement days |
| Monday | 9-10 AM | Decent but lower than Tue-Thu |
| Friday | 8-9 AM | Morning only — afternoon drops off |

### Format performance benchmarks

| Format | Avg Engagement Rate | Best For | Notes |
|--------|-------------------|----------|-------|
| Text posts | Baseline | Storytelling, hot takes, personal stories | Most resilient format, works across all algorithm changes |
| Carousels/Documents | 45.85% above baseline | Frameworks, step-by-step, educational | Highest engagement format in 2025-2026 |
| Photo posts | 20-30% above baseline | Milestone posts, behind-the-scenes | Authenticity matters — avoid stock photos |
| Video (native) | Variable | Clips, demos, talking head | Strong for retention but production overhead |
| Polls | Declining | Quick engagement | Algorithm has deprioritized polls |

### Reach killers

| Action | Impact | Alternative |
|--------|--------|------------|
| External links in body | -25-40% reach | First comment link |
| Editing post after publishing | Algorithm reset | Preview carefully before posting |
| Hashtags (3+) | Minimal/negative impact | 0-2 hashtags maximum |
| Posting multiple times/day | Second post cannibalizes first | 1 post/day maximum, 3-5x/week |
| Tagging people who don't engage | Signals low relevance | Only tag people likely to respond |

## YouTube optimization

### Algorithm priorities

| Signal | Weight | What It Means |
|--------|--------|--------------|
| **Watch time** | Highest | Total minutes watched — longer videos that retain win |
| **Click-through rate (CTR)** | Very high | Title + thumbnail drive 80% of clicks |
| **Audience retention** | High | % of video watched — front-loaded content is critical |
| **Suggested views** | Highest traffic source | Optimize for "related to" other videos in your niche |
| **Search views** | Second traffic source | SEO in title, description, and tags |

### Optimization checklist

- [ ] **Title:** SEO-keyword first, under 60 characters, not clickbait
- [ ] **Thumbnail:** High-contrast, readable text (if any), face with expression, A/B test
- [ ] **First 30 seconds:** Hook immediately — state the promise, show a result, or ask a question
- [ ] **Description:** First 2 lines appear above fold — make them count. Include timestamps, links, keywords
- [ ] **Tags:** 5-15 relevant keywords, mix broad and specific
- [ ] **End screen:** Subscribe button + related video
- [ ] **Chapters:** Use timestamps to create chapters — improves retention and SEO
- [ ] **Consistent schedule:** Same day/time each week builds audience habit

### Video length guidelines

| Type | Optimal Length | Notes |
|------|---------------|-------|
| Tutorial/how-to | 8-15 minutes | Long enough to be thorough, short enough to retain |
| Interview/podcast | 20-45 minutes | Cut to the best segments, ruthless editing |
| Shorts/clips | 30-60 seconds | One idea, one payoff. Native feeling, not obviously clipped |
| Product demo | 3-8 minutes | Show the result first, then the process |

## Newsletter optimization

### Key metrics and targets

| Metric | Good | Great | Excellent |
|--------|------|-------|-----------|
| **Open rate** | 30-40% | 40-50% | 50%+ |
| **Click-through rate** | 2-4% | 4-7% | 7%+ |
| **Subscriber growth** | 5-10%/month | 10-20%/month | 20%+ |
| **Unsubscribe rate** | < 0.5% | < 0.3% | < 0.1% |

### Optimization checklist

- [ ] **Subject line:** Under 50 characters, specific, curiosity-driven. A/B test when possible
- [ ] **Preview text:** Intentional — not auto-generated from first paragraph
- [ ] **Send time:** Tuesday or Thursday morning (8-10 AM in audience timezone) for B2B
- [ ] **UTM parameters:** Every link tagged for attribution tracking
- [ ] **Exclusive content:** Give subscribers something they can't get elsewhere
- [ ] **List hygiene:** Remove inactive subscribers quarterly (no opens in 90 days)
- [ ] **Reply-to:** Use a real email address — replies signal engagement to ESPs
- [ ] **Mobile optimization:** 60%+ of opens are mobile — test on phone

### Subject line patterns that work

| Pattern | Example | Why It Works |
|---------|---------|-------------|
| Number + specific | "3 homepage mistakes that kill conversions" | Concrete, scannable promise |
| Question | "Is your positioning actually working?" | Creates curiosity gap |
| Contrarian | "Stop writing content calendars" | Provokes click to understand |
| Result-first | "How [Company] 3x'd their demo rate" | Outcome signals value |
| Exclusive | "The framework I shared with zero clients" | Scarcity drives opens |

## Podcast optimization

### Key metrics and targets

| Metric | Good | Great | Notes |
|--------|------|-------|-------|
| **Downloads per episode** | 100-500 | 500-2,000 | Highly niche-dependent |
| **Listener retention** | 50%+ at halfway | 70%+ at halfway | Front-load the best content |
| **Subscriber growth** | Consistent weekly growth | 5-10% month-over-month | Focus on consistency |

### Optimization checklist

- [ ] **Episode title:** SEO-searchable keywords, not clever puns. "[Topic]: [Specific angle]"
- [ ] **First 2 minutes:** Cold open with the best insight. No 60-second intro music or throat-clearing
- [ ] **Show notes:** Include timestamps, key quotes, links, and guest info
- [ ] **Consistent schedule:** Same day/time builds listener habit
- [ ] **Cross-promotion:** Make it easy for guests to share — provide clips, quotes, graphics
- [ ] **Distribution:** Submit to Apple, Spotify, YouTube, Google Podcasts at minimum
- [ ] **Derivative extraction:** Plan clips, quotes, and graphics before recording — know what you're harvesting

### Episode structure template

```
[0:00] Cold open — best insight or quote from the episode (30 seconds)
[0:30] Brief intro — who, what, why this matters (60 seconds)
[1:30] Guest intro — who they are, what they've built (2 minutes)
[3:30] Core discussion — the meat of the episode (20-30 minutes)
[~30:00] Key takeaways — 2-3 actionable things the audience can do (3 minutes)
[~33:00] Where to find the guest — plugs, links, calls to action (1 minute)
[~34:00] Outro — quick sign-off (30 seconds)
```

## Lead capture methods

### Method 1: Engagement tracking

| Detail | Value |
|--------|-------|
| **What it captures** | Post likers, commenters, sharers |
| **Signal strength** | Medium (shows interest, not intent) |
| **Recommended tools** | Trigify, Valley, Phantombuster |
| **Data pull frequency** | Daily or weekly |
| **Data flow** | Tool → CSV/API → Clay → CRM |
| **Setup effort** | Low (connect LinkedIn account, configure filters) |

### Method 2: Profile views

| Detail | Value |
|--------|-------|
| **What it captures** | People who viewed your LinkedIn profile |
| **Signal strength** | High (active research behavior) |
| **Recommended tools** | Sales Navigator |
| **Data pull frequency** | Daily |
| **Data flow** | Sales Navigator → Manual or API → Clay → CRM |
| **Setup effort** | Low (requires Sales Navigator subscription) |

### Method 3: Lead magnets

| Detail | Value |
|--------|-------|
| **What it captures** | Email opt-ins from gated content |
| **Signal strength** | High (exchange of value) |
| **Recommended tools** | beehiiv, ConvertKit, LinkedIn Featured section |
| **Data pull frequency** | Real-time (automated) |
| **Data flow** | Form → Email platform → Zapier/API → Clay → CRM |
| **Setup effort** | Medium (create lead magnet, setup form, configure automation) |

### Method 4: Connection downloads

| Detail | Value |
|--------|-------|
| **What it captures** | New LinkedIn connections by week |
| **Signal strength** | Low-Medium (may be passive) |
| **Recommended tools** | LinkedIn export (Settings → Data Privacy → Get a copy of your data) |
| **Data pull frequency** | Weekly or biweekly |
| **Data flow** | LinkedIn export → CSV → Clay → CRM |
| **Setup effort** | Low (manual export) |

### Method 5: Website visits

| Detail | Value |
|--------|-------|
| **What it captures** | Identified companies/people visiting your website |
| **Signal strength** | High (active research behavior) |
| **Recommended tools** | RB2B, Clearbit Reveal |
| **Data pull frequency** | Daily (automated) |
| **Data flow** | Pixel → Tool dashboard → API/Zapier → Clay → CRM |
| **Setup effort** | Medium (install pixel, configure identification rules) |

### Method 6: Direct bookings

| Detail | Value |
|--------|-------|
| **What it captures** | Calendar bookings from content |
| **Signal strength** | Very High (explicit intent) |
| **Recommended tools** | Cal.com, Calendly |
| **Data pull frequency** | Real-time (automated) |
| **Data flow** | Booking → Calendar tool → Zapier → CRM |
| **Setup effort** | Low (add booking link to profile Featured section, post CTAs) |

## Enrichment flow

### Clay enrichment pipeline

```
Capture source (any of the 6 methods above)
  │
  ├─ Import to Clay table (manual CSV or automated via API/Zapier)
  │
  ├─ Enrich:
  │   ├─ Company: size, revenue, industry, tech stack, funding
  │   ├─ Person: title, department, LinkedIn URL, email
  │   └─ Custom: any enrichment columns relevant to ICP
  │
  ├─ Score:
  │   ├─ ICP fit score (company size + industry + title match)
  │   ├─ Engagement depth (comments > likes, multiple touches > single)
  │   └─ Intent signals (profile view + website visit = high intent)
  │
  ├─ Tag:
  │   ├─ Content source (which post/series drove the capture)
  │   ├─ Capture method (engagement, profile view, lead magnet, etc.)
  │   └─ Funnel stage (TOFU engager, MOFU downloader, BOFU booker)
  │
  └─ Sync to CRM:
      ├─ HubSpot: Create/update contact with tags and scores
      ├─ Salesforce: Create/update lead with attribution
      └─ Trigger downstream action based on score threshold
```
