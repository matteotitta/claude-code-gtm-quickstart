## Formatting Rules

1. **Tables:** Use for all quantitative data (metrics, scores, inventories)
2. **Percentages:** Always calculate and show actual percentages
3. **Benchmarks:** Include B2B benchmarks for context
4. **Examples:** Reference specific content pieces, not generic descriptions
5. **Scores:** Use 1-5 scale with clear definitions
6. **Gaps:** Be specific about what's missing and why it matters
7. **Recommendations:** Prioritize by effort/impact, make actionable
