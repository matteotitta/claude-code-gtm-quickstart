# Content audit — per-phase walkthrough

Full step-by-step for each of the six audit phases. Hoisted from the pre-slim SKILL.md (`Process (Step-by-Step)` section, lines 187–451). Each phase ends with a checkpoint to verify before proceeding.

---

## Phase 1: Channel Discovery & Setup

**Purpose:** Identify all active content channels and establish audit scope.

**Steps:**

1. **Step 1.1: Discover content channels**
   - Fetch website footer and navigation for links
   - Search: `site:[domain] blog OR newsletter OR resources`
   - Search: `"[company]" site:linkedin.com`
   - Search: `"[company]" site:youtube.com`
   - Search: `"[company]" site:lu.ma OR site:eventbrite.com`
   - **Output:** Channel inventory with URLs

2. **Step 1.2: Confirm channel access**
   - Verify each URL is accessible
   - Note any API access available
   - Identify rate limits or access restrictions
   - **Output:** Confirmed channels with access level

3. **Step 1.3: Establish audit scope**
   - Confirm time range (default: last 6 months)
   - Prioritize channels by apparent activity
   - Note any channels to skip (inactive, restricted)
   - **Output:** Scoped audit plan

**Phase 1 Checkpoint:**
- [ ] All channels discovered
- [ ] Access confirmed for each
- [ ] Audit scope defined

## Phase 2: Newsletter/Blog Audit

**Purpose:** Analyze long-form content performance, topics, and consistency.

**Steps:**

1. **Step 2.1: Fetch newsletter/blog content**
   - Fetch archive/index page
   - Extract post list with dates and titles
   - Sample 10-15 recent posts for deep analysis
   - **Output:** Content inventory

2. **Step 2.2: Analyze publishing cadence**
   - Calculate posts per week/month
   - Identify consistency patterns
   - Note gaps or bursts
   - **Output:** Cadence analysis

3. **Step 2.3: Analyze content topics**
   - Extract topic themes from titles and content
   - Categorize by content pillar (Educational/Personal/Promotional/Organizational)
   - Identify topic clusters
   - **Output:** Topic analysis

4. **Step 2.4: Analyze content formats**
   - Identify format types (guides, tutorials, case studies, opinion, etc.)
   - Assess format mix
   - Note unique formats or series
   - **Output:** Format analysis

5. **Step 2.5: Assess content quality signals**
   - Analyze headline patterns
   - Check for CTAs and lead magnets
   - Assess depth and structure
   - **Output:** Quality assessment

**Phase 2 Checkpoint:**
- [ ] Content inventory complete
- [ ] Cadence analyzed
- [ ] Topics categorized
- [ ] Formats assessed
- [ ] Quality signals noted

## Phase 3: LinkedIn Audit

**Purpose:** Analyze social content performance, engagement, and voice.

**Steps:**

1. **Step 3.1: Fetch LinkedIn content**
   - Access company page or founder profile
   - Extract recent posts (last 20-30 posts)
   - Note: Full metrics require API or manual review
   - **Output:** Post inventory

2. **Step 3.2: Analyze posting cadence**
   - Calculate posts per week
   - Identify posting patterns (days, times)
   - Note consistency
   - **Output:** Cadence analysis

3. **Step 3.3: Analyze content types**
   - Categorize: Text, carousel, video, image, poll, article
   - Assess mix ratios
   - Identify top-performing formats (if visible)
   - **Output:** Format analysis

4. **Step 3.4: Analyze topics and themes**
   - Extract topic patterns
   - Categorize by pillar
   - Identify recurring series or themes
   - **Output:** Topic analysis

5. **Step 3.5: Assess engagement patterns**
   - Note visible engagement (likes, comments)
   - Identify high-performing content patterns
   - Assess comment quality (community vs. spam)
   - **Output:** Engagement assessment

6. **Step 3.6: Analyze voice and hooks**
   - Sample 5-10 posts for voice analysis
   - Identify hook patterns
   - Assess CTA usage
   - **Output:** Voice analysis

**Phase 3 Checkpoint:**
- [ ] Post inventory complete
- [ ] Cadence analyzed
- [ ] Formats categorized
- [ ] Topics mapped
- [ ] Engagement assessed
- [ ] Voice patterns noted

## Phase 4: YouTube Audit (If Applicable)

**Purpose:** Analyze video content strategy, performance, and production quality.

**Steps:**

1. **Step 4.1: Fetch channel data**
   - Access channel page
   - Extract subscriber count, video count
   - List recent videos with titles and dates
   - **Output:** Channel overview

2. **Step 4.2: Analyze publishing cadence**
   - Calculate videos per month
   - Identify consistency patterns
   - Note series or playlists
   - **Output:** Cadence analysis

3. **Step 4.3: Analyze content types**
   - Categorize: Tutorials, interviews, product demos, vlogs, shorts
   - Assess format mix
   - Note video lengths
   - **Output:** Format analysis

4. **Step 4.4: Assess visible performance**
   - Note view counts on recent videos
   - Identify high-performing content
   - Assess thumbnail and title patterns
   - **Output:** Performance assessment

5. **Step 4.5: Analyze production quality**
   - Sample 2-3 videos for quality check
   - Assess: Audio, visuals, editing, branding
   - Note production level (DIY vs. professional)
   - **Output:** Production assessment

**Phase 4 Checkpoint:**
- [ ] Channel data fetched
- [ ] Cadence analyzed
- [ ] Content types categorized
- [ ] Performance assessed
- [ ] Production quality noted

## Phase 5: Events Audit (If Applicable)

**Purpose:** Analyze events content strategy and community engagement.

**Steps:**

1. **Step 5.1: Fetch events data**
   - Access Luma, Eventbrite, or events page
   - List recent and upcoming events
   - Extract event types and topics
   - **Output:** Events inventory

2. **Step 5.2: Analyze event cadence**
   - Calculate events per month/quarter
   - Identify event types (webinar, workshop, conference, meetup)
   - Note virtual vs. in-person mix
   - **Output:** Cadence analysis

3. **Step 5.3: Assess event topics and themes**
   - Categorize event topics
   - Map to content pillars
   - Identify recurring series
   - **Output:** Topic analysis

4. **Step 5.4: Note community signals**
   - Assess attendance visibility (if shown)
   - Note speaker patterns (solo, guests)
   - Check for event recordings/repurposing
   - **Output:** Community assessment

**Phase 5 Checkpoint:**
- [ ] Events inventory complete
- [ ] Cadence analyzed
- [ ] Topics categorized
- [ ] Community signals noted

## Phase 6: Cross-Channel Synthesis

**Purpose:** Compile findings into actionable audit output.

**Steps:**

1. **Step 6.1: Calculate content volume**
   - Total pieces per month across channels
   - Per-channel breakdown
   - Compare to B2B benchmarks
   - **Output:** Volume summary

2. **Step 6.2: Assess content pillar mix**
   - Aggregate pillar ratios across channels
   - Compare to recommended 40/25/25/10
   - Identify imbalances
   - **Output:** Pillar mix analysis

3. **Step 6.3: Identify content gaps**
   - Missing pillars or formats
   - Underserved topics
   - Channel gaps (e.g., no video)
   - **Output:** Gap analysis

4. **Step 6.4: Assess content consistency**
   - Cross-channel voice consistency
   - Topic alignment
   - Brand consistency
   - **Output:** Consistency assessment

5. **Step 6.5: Identify top-performing patterns**
   - Best content formats per channel
   - Successful topic themes
   - Effective hook/headline patterns
   - **Output:** Success patterns

6. **Step 6.6: Generate recommendations**
   - Quick wins (low effort, high impact)
   - Strategic gaps to fill
   - Channel-specific improvements
   - Content cascade opportunities
   - **Output:** Recommendations

7. **Step 6.7: Assign audit scores**
   - Volume score (1-5)
   - Consistency score (1-5)
   - Quality score (1-5)
   - Gap score (1-5, inverse)
   - Overall content health score
   - **Output:** Audit scorecard

**Phase 6 Checkpoint:**
- [ ] Volume calculated
- [ ] Pillar mix assessed
- [ ] Gaps identified
- [ ] Consistency assessed
- [ ] Success patterns noted
- [ ] Recommendations generated
- [ ] Scores assigned
