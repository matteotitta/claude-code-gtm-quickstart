# Webinar format templates

Pick a format based on goal and time budget.

## Format selection matrix

| Format | Duration | Best For |
|--------|----------|----------|
| Solo presentation | 30-45 min | Thought leadership |
| Panel discussion | 45-60 min | Multiple perspectives |
| Interview | 30-45 min | Customer/expert spotlight |
| Demo + Q&A | 30-45 min | Product-focused |
| Workshop | 60-90 min | Hands-on learning |

## 45-Minute Thought Leadership Webinar

| Time | Section | Content |
|------|---------|---------|
| 0:00-5:00 | Opening | Hook, agenda, speaker intro |
| 5:00-15:00 | Problem | Current state, why it matters |
| 15:00-35:00 | Insights | 3-5 key points with examples |
| 35:00-40:00 | Application | How to implement |
| 40:00-45:00 | Q&A + Close | Questions, next steps |

## 60-Minute Panel Discussion

| Time | Section | Content |
|------|---------|---------|
| 0:00-5:00 | Opening | Topic intro, panelist intros |
| 5:00-15:00 | Question 1 | Each panelist responds |
| 15:00-25:00 | Question 2 | Each panelist responds |
| 25:00-35:00 | Question 3 | Each panelist responds |
| 35:00-45:00 | Audience Q&A | Moderated questions |
| 45:00-55:00 | Lightning round | Quick takes |
| 55:00-60:00 | Close | Key takeaways, next steps |

## 30-Minute Demo Webinar

| Time | Section | Content |
|------|---------|---------|
| 0:00-3:00 | Opening | Problem statement, agenda |
| 3:00-8:00 | Context | Who this is for, use case |
| 8:00-23:00 | Demo | Live product walkthrough |
| 23:00-28:00 | Q&A | Audience questions |
| 28:00-30:00 | Close | Offer, next steps |

## Default 45-min content allocation

| Section | Time | Content |
|---------|------|---------|
| Intro | 5 min | Hook, agenda, speakers |
| Context | 10 min | Problem/opportunity framing |
| Core content | 20 min | Main insights/demo |
| Application | 5 min | How to apply |
| Q&A | 5 min | Audience questions |
