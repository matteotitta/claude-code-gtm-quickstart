# competitors/

One file per competitor + an aggregate insights doc that synthesizes patterns across them.

**What goes here:**
- `MMYY-{competitor-slug}.md` — full 11-dimension competitor analysis (positioning, pricing, features, GTM, content, etc.)
- `MMYY-aggregate-insights.md` — patterns across the full competitor set (common positioning angles, pricing gaps, white space, threat tiers)
- Update individual competitors quarterly (or when you lose a deal to them); aggregate after each batch

**Produced by:** `/competitor-research`

**Depends on:** company-context (your own firmographics — for accurate fit comparisons).

**Feeds:** positioning, messaging, sales deck, battlecards (gated), win-loss analysis (gated).
