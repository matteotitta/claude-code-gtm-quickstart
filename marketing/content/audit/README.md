# content/audit/

Outputs that **assess** the current state of your content. Reports, scores, gap analyses.

**What goes here:**
- `MMYY-content-audit.md` — full content inventory + scoring (from `/content-audit`)
- Optional: `MMYY-channel-audit-{channel}.md` — single-channel deep dives (e.g., LinkedIn, blog)

**Produced by:** `/content-audit`

**Naming:** `MMYY-topic.md` (e.g., `0526-content-audit.md`).

**Read before producing content strategy** — your strategy should be informed by what's already working / not working.

**Feeds:** content strategy, editorial calendar, prioritization decisions.
