# content/execution/

Outputs that **produce + ship** content. The actual writing — long-form articles, thought leadership, drafts.

**What goes here:**
- `MMYY-thought-leadership-{topic}.md` — long-form pieces from `/thought-leadership`
- `MMYY-draft-{title}.md` — work-in-progress drafts
- `MMYY-published-{title}.md` — final shipped versions (rename when published)

**Produced by:** `/thought-leadership`

**Naming:** `MMYY-{type}-{topic}.md` (e.g., `0526-thought-leadership-zero-outbound.md`).

**Depends on:** content strategy (what to write), expert POV / voice (how to write), ICP + messaging (who it's for). Read those first.

**This is where the work ships from.** Audit measures, strategy plans, execution writes.
