# docs/

Briefs, contracts, transcripts, meeting notes, source material — anything that informs but isn't a skill output.

**What goes here:**
- Sales call transcripts (raw or cleaned)
- Customer interview transcripts
- Stakeholder meeting notes
- Strategy briefs from leadership
- Source decks, PDFs, reference materials
- Skill outputs that don't fit any other PMM-core folder (e.g., `MMYY-launch-{product}.md`, `MMYY-case-{customer}.md`, `MMYY-webinar-{topic}.md`)

**Naming:** `MMYY-topic.md` (e.g., `0526-q1-stakeholder-debrief.md`). For non-markdown source files, keep the original filename and add a `MMYY-source-` prefix if you want the date marker (e.g., `0526-source-pricing-deck.pdf`).
